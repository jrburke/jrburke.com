volo 0.2.5 released

Mon, 26 Nov 2012 19:48:18 GMT

This update for volo is mostly about working well with the new GitHub rate limit changes for unauthenticated requests. 0.2.5 will use GitHub auth tokens if the non-token API limit has been reached.

The other benefit with this approach, you can now pull down private repos via volo create and volo add!

It is strongly recommended you upgrade to 0.2.5:

npm install -g volo
