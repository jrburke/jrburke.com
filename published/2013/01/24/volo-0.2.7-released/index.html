volo 0.2.7 released

Fri, 25 Jan 2013 02:11:25 GMT

A maintenance release for volo that fixes some issues around branch fetching and proper ordering of version tags. List of fixed 0.2.7 issues.

Upgrading is recommended:

npm install -g volo

Also, if you have a GitHub repo that can be used as a volo create project template, feel free to add it to the create template directory.
